# Master Era HRM React Frontend Demo

React.js only frontend demo for Master Era HRM Cloud Software.

## Run in VS Code

```bash
npm install
npm run dev
```

## Demo Button Logic

- Employees > Add Employee: opens employee form. Submit karta employee table ma new row add thay.
- Topbar > New Request: opens HR request form. Submit karta notification list ma alert add thay.
- Topbar > Bell: shows demo notifications.
- Attendance > Month and Year: employee/month wise attendance search thay.
- Attendance > Original Data: filter reset kari current demo attendance batave.
- Leave > Apply Leave: employee leave request create thay and HR table ma pending ave.
- Leave > View: selected employee ni leave history and leave detail dekhashe.
- Leave > Approve/Reject: HR frontend state ma status update kare.
- Topbar > Employee Side: same URL ma employee self-service portal open thay.
- Employee Side pages: My Dashboard, My Profile, My Attendance, My Leave, My Payslip, My Documents.
- Employee Side > HR Side: pachu HR admin dashboard ma switch thay.

## Client Demo Line

Aa hamna only React frontend demo che. Ek j URL ma HR Side ane Employee Side banne client ne batavi shakay. Actual project ma aa forms, buttons, attendance filters, leave approvals, payroll and notifications backend API/database sathe connect thashe.
